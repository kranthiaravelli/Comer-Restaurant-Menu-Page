# Comer Restaurant Menu Page

## About  
Comer Restaurant Menu Page is a simple frontend landing page project created as a concept restaurant website. It showcases an online menu browsing idea and basic restaurant webpage layout.

This project was built using HTML and CSS as part of frontend development practice. It focuses on structure, styling, and basic UI design.

Note: The project is not responsive and is optimized for desktop view only.

## Technologies Used  
- HTML5  
- CSS3  
- GitHub (version control)

## Features  
- Restaurant landing page design  
- Menu layout structure  
- Simple and clean UI  
- Static concept-based booking page  

## Purpose  
This project was created to practice frontend fundamentals including HTML structure, CSS styling, and basic web page design.
